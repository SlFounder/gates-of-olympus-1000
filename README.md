# gates-of-olympus-1000
Gates of Olympus 1000 — усиленная версия легендарного слота от Pragmatic Play, где множители достигают x1000. Эта модификация делает игру ещё более динамичной и взрывной. Улучшенные каскады, частые бонусы и повышенный потенциал заноса позволяют поймать рекордные выигрыши даже с минимальной ставки.
